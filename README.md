#Instalação

***Requisitos(screenfetch e lolcat instalados***
	       
	        ```
		ArchLinux = sudo pacman -S screenfetch.
		Debian,Ubuntu... = sudo apt-get install screenfetch.
		ArchLinux = sudo pacman -S lolcat.
		Debian,Ubuntu... = sudo apt-get install lolcat.
		```
